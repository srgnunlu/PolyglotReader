# Corio Docs Brand Assets

The mark combines a folded document, a translation bubble, and a short
annotation stroke. It is intentionally free of sparkle, bot, and brain motifs.

Core palette:

- Ink `#182238`
- Coral `#F06A4D`
- Translation teal `#2F9C95`
- Paper `#F7F3EA`
- AI gold `#D99A2B`

`CorioDocs-AppIcon*.svg` files are the editable vector sources. The corresponding
1024 px PNG files in `Assets.xcassets/AppIcon.appiconset` are rendered from these
sources and must stay visually identical.
