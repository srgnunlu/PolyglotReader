# App Store Assets

`generate-assets.sh` composites real simulator captures into the accepted
1290×2796 iPhone 6.9-inch portrait format. The source captures intentionally
come from the network-free entry demo so the marketing set never invents a
feature or exposes user data.

Regenerate after a material entry-screen change:

```sh
Marketing/AppStore/generate-assets.sh
```

Final Turkish assets are written to `iPhone-6.9/tr/`. Keep the editable SVG
beside each PNG so copy and placement can be revised without degrading the
screen capture.
